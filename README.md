# Image Compressor

A simple Python tool to compress images quickly and efficiently.

This script compresses images from the `input_path` directory and creates an `Optimized_images` folder with the optimized files inside.

It helps reduce large images from around 30MB–60MB to much smaller sizes, often around 2MB–8MB, while preserving good visual quality.

## Features
- Batch image compression.
- Automatic output folder creation.
- Easy to use with a single input path.
- Designed to save storage and speed up uploads.

## Usage
1. Set the `input_path` variable to the folder containing your images.
2. Run the script.
3. Find the compressed images in the `Optimized_images` folder.

## Requirements
Install dependencies with:
```bash
pip install -r requirements.txt
```

## Notes
- Best for users who need lighter images for web, e-commerce, sharing, or storage.
- Output size depends on the original image format and quality settings.
